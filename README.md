To use this you will need (for the current config) waybar cava hyprpaper hyprlock rofi fastfetch hyfetch 
For screenshot functionalities you will need gim wl-clipboard and dunstify

To install this just copy the hypr into your .config/hypr and add hyplandThemes to your .config 

BEWARE most of the existing scripts are distructif so if you have a currently customised hyprland .conf file it will be voided by the scripts (some of them create stim links that will mess up existing files (this is the case in the exec parts of the hyprland.conf))
same goes for preexisting hyfetch and fastfetch config files others should be fine 

I would recomend adding your current configs to a folder in hyprlandThemes and modify paths inside the configuration just remember the conf iself need to be hyprland.conf or it may not work (check the themes.sh script in the script directory to see how it works)
